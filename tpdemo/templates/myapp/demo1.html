1.模板中的变量输出

  • {{ name }}
  • {{ stu.name }} {{ stu.age }}
  • {{ a.0 }}

2.模板中的标签输出

{% for var in dist %} {% endfor %}
姓名 性别 年龄 状态

token 令牌：{% csrf_token %} 单行注释: {# 注释 #} 多行注释: {% comment %} {% endcomment %}

3.模板中的过滤器 |

  • 原值: {{ name }}
  • 大写: {{ name|upper }}
  • 小写: {{ name|lower }}
  • {{ time|date:'Y-m-d H:i:s' }}
  • {{ a|join:":"}}

4.模板中的运算

  • 两个数: {{m1}}{{m2}}
  • 求和: {{m1|add:m2}}
  • 求差: {{m1|add:-10}}
  • 相乘: {% widthratio m1 1 m2 %}
  • 相除: {% widthratio m1 10 1 %}

5.自定义过滤器和模板标签

{% load pagetag %} 大写：{{ name|myupper }}
相减：{% jian m1 m2 %}
